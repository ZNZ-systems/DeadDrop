Live Demo

Try the Widget

Click the floating mail button in the bottom-right corner to open the contact form widget. This is exactly what your website visitors would see.

How to Embed

Add this single script tag to any HTML page:

<script
  src="http://localhost:8080/static/widget.js"
  data-deaddrop-id="YOUR_DOMAIN_UUID">
</script>

Replace YOUR_DOMAIN_UUID with the public ID from your registered domain in the DeadDrop dashboard.

How It Works

1. Register your domain in the dashboard
2. Verify ownership via DNS TXT record
3. Add the embed code to your site
4. Messages appear in your dashboard
