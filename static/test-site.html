Welcome to the Test Site

This page is a plain HTML site running on localhost:4000. It demonstrates how the DeadDrop widget integrates into any website. Look for the floating mail icon in the bottom-right corner — that's the DeadDrop contact form widget.

Widget Active

How It Works

The widget is loaded via a single script tag. It creates a shadow DOM overlay with a contact form that submits messages to the DeadDrop API. Domain owners can view and manage all submitted messages from their dashboard.
